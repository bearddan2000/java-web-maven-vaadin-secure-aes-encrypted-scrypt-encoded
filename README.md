# java-web-maven-vaadin-secure-aes-encrypted-scrypt-encoded

## Description
A vaadin login example using
AES encryption with scrypt
one-way hash.

## Tech stack
- java
- maven
  - gwt
  - aes
  - scrypt
  - vaadin

## Docker stack
- adoptopenjdk/maven-openjdk11

## To run
`sudo ./install.sh -u`
http://localhost
- username: user
- password: pass

## To stop (optional)
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
